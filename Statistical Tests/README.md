Free Statistical Courses for Statistical Consulting (Revision List)
================
Moses Mburu
2026-02-19

### 1) Regression and diagnostics (high ROI for consulting)

- [Penn State STAT 501: Regression Methods (free
  notes)](https://online.stat.psu.edu/stat501/)
- [UCLA IDRE: Regression Diagnostics in R (free
  tutorial)](https://stats.oarc.ucla.edu/r/dae/linear-regression/)
- [StatQuest (YouTube): Linear/Logistic Regression and intuition
  building](https://www.youtube.com/c/joshstarmer)

### 2) Core statistics foundations (inference + tests)

- [MIT OpenCourseWare 18.650: Statistics for Applications (videos +
  materials)](https://ocw.mit.edu/courses/18-650-statistics-for-applications-fall-2016/)
- [OpenIntro Statistics (free textbook +
  labs)](https://www.openintro.org/book/os/)
- [OpenStax Introductory Statistics 2e (free
  textbook)](https://openstax.org/details/books/introductory-statistics-2e)

### 3) Probability (for calm, correct reasoning)

- [Harvard STAT 110 (Blitzstein) — full YouTube
  playlist](https://www.youtube.com/playlist?list=PL2SOU6wwxB0uwwH80KTQ6ht66KWxbzTIo)

### 4) Clinical trials and randomisation (consulting-adjacent skills)

- [FutureLearn: Introduction to Randomised Controlled Trials (University
  of
  Birmingham)](https://www.futurelearn.com/courses/randomised-controlled-trials)

### 5) Access notes (how to use platforms for free)

- [edX: What is the audit track? (Free access to course materials may be
  available)](https://help.edx.org/edxlearner/s/article/What-is-the-audit-track)
- [Class Central: Coursera “free” enrollment/audit options (varies by
  course)](https://www.classcentral.com/report/coursera-signup-for-free/)

### Suggested study workflow (consulting mode)

For each topic you study, write short notes answering:

1.  **When to use it**
2.  **Key assumptions**
3.  **Diagnostics / checks**
4.  **How you would explain results to a non-technical stakeholder**
5.  **Common failure modes and what you do instead**
